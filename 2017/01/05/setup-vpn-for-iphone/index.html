setup-vpn-for-iphone

发表于 2017-01-05 | 分类于 Other

半年前就在搬瓦工上买的服务器，用来翻墙，和挂几个没有备案的域名，体验还不错，国内访问速度也不错，ping平均能有180ms左右
支持控制面板一键安装Shadowsocks和OpenVPN等常用翻墙工具，特别方便

不过之前只是用的Shadowsocks配合SwichSharp在Chrome上用，很好用，一只没折腾过手机(iPhone)翻墙

今天有需要用到手机翻墙，还挺多坑的，这里纪录一下

基本的步骤可以看这里

有几个需要注意的问题：

  1. OpenVZ还是Xen、KVM 这个选错了会导致连接不成功，我刚开始选的Xen、KVM一直失败，后来选择OpenVZ才能成功连上
  2. 可能是因为我之前安装了OpenVPN，导致系统中有两个版本的ipsec，所以安装完以后，发现ca.cert.pem文件是空的。这里需要删掉旧版的ipsec
  3. 网卡的选择这个也要注意，选错了会导致能连上VPN，但是不能上网，我之前一直以为应该用显示外网ip的那个venet0:0，后来换了venet0才可以
  4. 记得把ca.cert.pem的后缀改成cer的
  5. 重新安装之前记得先ipsec stop

anyway,重装了4 5遍，终于可以翻墙了
