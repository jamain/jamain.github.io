发表于 2017-01-01 | 分类于 Other

这次博客折腾的，用起来相对比较顺手了，记录一下遇到的问题

Hexo

之前用jekyll的时候，总是有莫名其妙的编不过的问题，偶然发现Hexo，发现使用起来更简单
感觉hexo的原理就是，在jekyll上面封装了一层

hexo init 是创建一个hexo的工作环境目录，之前jekyll那部分东西，其实是在.deploy_git里面了

hexo每次发布，会先生成对应的发布文件，然后再将生成后的文件push到git

主题我是用的next

标签页，分类页，社交信息等设置参见这里

Markdown

写代码比较习惯Sublime，写Markdown可以安装MarkdownEditing插件
比较常用的快捷键有：

拷贝一个链接，选中文本后按下 ⌘⌥V 会自动插入内联链接。
拷贝一个链接，选中文本后按下 ⌘⌥R 会自动插入引用链接。

还可以安装OmniMarkupPreviewer插件进行预览，或者导出html文件
